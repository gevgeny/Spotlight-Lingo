Introduction

This Privacy Policy outlines how Spotlight Lingo handles user data.

Data Collection

The extension itself does not collect any personal information or data from its users. We do not store, share, or use any personally identifiable information. However, we do collect the following anonymized data for analytics purposes:
  1. Anonymized User ID: A randomly generated identifier that does not allow for personal identification.
  2. Site Information: The site where the extension is active at the moment the event is sent.
  3. Translation Languages: The source translation language and the target translation language selected by the user.
  4. Browser Model and Version: Information about the user's browser type and version.
  5. Browser Timezone: The timezone set in the user's browser.
  6. Operating System: The operating system the user is running.
This data is used solely for understanding how the extension is being used and to improve our service. All collected data is anonymous and cannot be used to identify individual users.

Compliance with Regulations

Our data collection practices ensure compliance with applicable privacy regulations, including GDPR and other relevant privacy laws. Since we only process anonymized data, no personal data is transferred or stored outside the EU unless required to provide services to non-EU customers.

Changes to this Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page, and the "Last Updated" date will be revised accordingly.
